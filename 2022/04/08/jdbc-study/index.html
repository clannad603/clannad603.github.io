java database connectivity(java 语言连接数据库)

jdbc本质为接口，在具体的实现上由各大数据库完成，面向接口编程，确保规范一致

Jdbc编程步骤

  1. 注册驱动(告诉java程序，即将执行的是那个品牌的数据库)
  2. 获取连接
  3. 获取数据库操作对象(专门执行sql语句的对象)
  4. 执行sql语句
  5. 处理查询结果集
  6. 释放资源
